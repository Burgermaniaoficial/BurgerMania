# Burger Mania Website

Um site moderno e impactante para a hamburgueria Burger Mania, focado em gerar desejo e conversão para pedidos.

## Estrutura do Projeto

- `index.html` - Página principal do site
- `styles.css` - Estilos CSS responsivos e modernos
- `script.js` - JavaScript para interações e animações

## Como Visualizar

1. Abra o arquivo `index.html` em um navegador web moderno
2. Ou execute um servidor local:
   - Python: `python -m http.server 8000`
   - Node.js: `npx serve .`
   - Abra `http://localhost:8000` no navegador

## Características

- Design escuro e premium
- Mobile-first e responsivo
- Animações suaves e efeitos visuais
- Foco em conversão com botões de pedido
- Imagens de alta qualidade (placeholders de Unsplash)

## Tecnologias

- HTML5
- CSS3 (Flexbox, Grid, Animações)
- JavaScript (ES6+)
- Google Fonts (Oswald, Roboto)

## Personalização

Para alterar imagens, substitua as URLs no `index.html` por suas próprias imagens.
Para modificar cores, edite as variáveis no `styles.css`.
